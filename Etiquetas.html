Las etiquetas de encabezado son elementos en el código HTML de tu sitio que representan los encabezados en tu página. Cada página de tu sitio tiene que tener su propio conjunto único de etiquetas de encabezado. Usa diferentes tipos de etiquetas de encabezados como tus títulos,
para estructurar tu página y dividirla en secciones y subsecciones. Los encabezados ayudan a los visitantes de tu sitio a comprender de qué se trata cada página y pueden ayudar a la optimización para motores de búsqueda de tu sitio. Importante:Las etiquetas de encabezado también son importantes para la accesibilidad de tu sitio, y cómo presentar el contenido. Obtén más información sobre accesibilidad.

La etiqueta de encabezado H1 actúa como el título de cada página. Esta etiqueta es el titular de la página, parecido al titular en la página de un periódico.

Puedes cambiar la jerarquía de la etiqueta del encabezado (por ejemplo de ) sin afectar el contenido o diseño. Esto puede ser útil si necesitas reestructurar tu página mientras mantienes el aspecto de tu texto igual.

Para utilizar correctamente las etiquetas de encabezado HTML sigue alguna de estas prácticas: evita omitir niveles de encabezado HTML, evita utilizar el más de una vez en una página. Tampoco debes utilizar niveles inferiores para reducir el tamaño de la fuente, para esto, aplica la propiedad CSS font-size.
